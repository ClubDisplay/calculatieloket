# Project Atlas — Documentatie

> Laatst bijgewerkt: 2026-06-27

## Doel

Project Atlas is de gestructureerde documentatie-kern van Calculatieloket.nl.  
Deze map bevat het volledige naslagwerk voor ontwikkeling, SEO, content en ontwerp van de site.

## Hoe deze documentatie te gebruiken

- **Ontwikkelaars** starten bij `01-BLUEPRINT.md` en `03-DEVELOPMENT-STANDARDS.md`.
- **SEO-specialisten** gebruiken `02-SEO-PLAYBOOK.md`.
- **Contentmanagers** volgen `06-CONTENT-STANDARDS.md`.
- **Ontwerpers** raadplegen `09-BRAND-GUIDE.md` en `04-COMPONENT-LIBRARY.md`.
- **AI-assistenten** lezen `08-AI-AGENTS.md` voor context en instructies.
- **Projectmanagers** bekijken `00-MASTER-ROADMAP.md` voor de planning.
