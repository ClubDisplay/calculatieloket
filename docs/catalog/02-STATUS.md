## Status-definities

| Status | Betekenis |
|--------|-----------|
| `planned` | Gepland maar nog niet gestart |
| `draft` | In ontwikkeling (lokaal, niet live) |
| `review` | Klaar voor review door Barry |
| `live` | Live op productie |
| `deprecated` | Uitgefaseerd, niet meer actief onderhouden |
| `retired` | Verwijderd van de site |
